# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

Only the latest version receives security updates.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, report them via GitHub's private security advisory feature:

1. Go to the **Security** tab of this repository
2. Click **"Report a vulnerability"**
3. Fill in the details

Alternatively, email the maintainer directly.

### What to expect

- **Acknowledgment** within 48 hours
- **Status update** within 7 days
- If confirmed, a fix will be prioritized and you'll be credited (unless you prefer anonymity)

## Scope

This policy covers the plugins in this repository. Issues with Claude Code itself should be reported to Anthropic directly.
